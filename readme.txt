== Description ==
 
Ticker Pro probably the simplest and lightweight ticker in the world. Which display your ticker ony any page or post.
 

== Installation ==
 
1. Unzip and Upload 'ticker-pro' folder to the '/wp-content/plugins/' directory
    or
    simply add the plugin from WordPress Admin Menu
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Copy this  Shortcode to your post or page content box
    [ticker-pro]
4. To change the Ticker title add shortcode like , 
    [ticker-pro tickertitle='Notice :']
5.You can Add and Delete Tickers from ticker menu.
